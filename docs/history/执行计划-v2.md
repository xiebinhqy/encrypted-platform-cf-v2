```
✅ 注册流程：登录密码 → 恢复码 → 设置解密密码/锁屏密码
❌ 登录后"加密模式"显示密文，右上角有解密按钮（UI 未实现，数据层已完成）
❌ 输入解密密码后进入"解密模式"，正常显示明文（dialog 未实现）
❌ Ctrl+L 锁定 → 输入锁屏密码 → 直接恢复解密模式（auto-lock 实现了但验证的是 masterKey）
❌ 忘记密码 → 恢复码验证 → 重置密码（login.js 使用了旧版 recover API）
❌ 加密模式下所有增删改查操作被禁用（未实现）
✅ 密文显示格式："f8a2b7c9..." 截断（data-loader 已实现）
```

## 🔍 已完成的新功能

### 功能 1：备份与恢复（设置 → 备份与恢复）
- **导出全部数据**：点击后收集 AppState 中的笔记和分类，生成标准 JSON 文件下载，包含版本号、时间戳、摘要信息
- **清理缓存**：清除 IndexedDB 缓存 + localStorage 草稿 + sessionStorage 解密密钥，保留登录态，自动重新加载数据
- **导入数据**：通过隐藏 file input 选择 JSON 备份文件，解析验证格式后合并到 AppState，支持笔记 ID 匹配替换/去重

### 功能 2：亮色主题全面美化（v2.2.0）
- CSS 变量体系重构，新增 `--color-heading`、`--nav-active-bg`、`--nav-hover-bg` 等变量
- 白色主背景 + slate-50 次级背景 + slate-900 文字色
- 全面覆盖：标题/文字/卡片/输入框/侧边栏/表格/开关/编辑器/日志等组件
- Header 毛玻璃效果（backdrop-filter: blur(12px)）
