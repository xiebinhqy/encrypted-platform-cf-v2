## 📊 当前问题诊断

| 问题类型 | 具体表现 | 根因分析 |
|----------|----------|----------|
| **首次加载慢** | 登录后仪表盘加载慢 | Worker 冷启动 + D1 冷读取 + 全量解密 |
| **操作卡顿** | 创建分类/笔记/标签不流畅 | 每次操作都重新加载全量数据 |
| **页面切换慢** | 现代↔经典切换慢 + 跳登录页 | 两套 UI 是独立 Worker，认证状态不共享 |
| **数据不互通** | 一端创建另一端看不到 | 独立 Worker + 独立数据库连接 |
