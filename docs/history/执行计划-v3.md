## 🔍 备份与恢复功能详情

### 功能 1：导出全部数据
- 位置：设置 → 备份与恢复 → 导出全部数据
- 逻辑：收集 AppState 中所有笔记 + 分类数据，构建标准 JSON 格式
- 格式：`{ version, exportedAt, userId, summary, data: { notes[], categories[] } }`
- 下载：通过 Blob + URL.createObjectURL 触发浏览器下载，文件名含时间戳

### 功能 2：清理缓存
- 位置：设置 → 备份与恢复 → 清理缓存
- 清理范围：
  - IndexedDB 缓存（note-cache.js 的 clearAllCache）
  - localStorage 草稿（encrypted_notes_drafts）
  - localStorage 编辑器状态（encrypted_notes_current_editor）
  - sessionStorage 解密密钥（decryptKey）
- 保留内容：登录态（userId/masterKey/authToken 不变）
- 后续操作：自动调用 DataLoader.loadAll() 重新加载数据

### 功能 3：导入数据
- 位置：设置 → 备份与恢复 → 选择文件导入
- 支持格式：通过导出功能生成的 JSON 备份文件
- 合并逻辑：
  - 笔记：按 ID 匹配，已存在则替换，不存在则追加
  - 分类：按 ID 匹配去重
  - 导入完成后自动刷新仪表盘和数据
