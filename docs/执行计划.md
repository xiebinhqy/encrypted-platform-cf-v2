## 🔍 Docker 部署架构详情

### 架构
```
用户浏览器 → Docker 容器
               ├── Nginx/Express 静态文件 → frontend/modern/
               ├── Node.js API 服务 → backend/src/
               └── better-sqlite3 数据库 → /app/data/notes.db（持久化卷）
```

### 镜像配置
- **基础镜像：** node:20-alpine
- **多架构：** linux/amd64 + linux/arm64
- **多阶段构建：** 构建阶段 → 运行阶段（减少体积）
- **健康检查：** `wget --spider http://localhost:3000/api/health`

### 数据持久化
- SQLite 数据库文件存储在 Docker volume `notes-data` 中
- 重启容器数据不丢失
- 可通过 `docker compose down -v` 清除

### 一键启动（Windows）
```batch
start-docker.bat
```
脚本自动：检测 Docker 环境 → 生成 .env 文件（如不存在） → docker compose build → docker compose up -d
